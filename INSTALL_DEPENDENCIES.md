# Add these dependencies to your package.json:

```json
{
  "dependencies": {
    "googleapis": "^128.0.0",
    "pusher-js": "^8.4.0-rc2",
    "tiktok-live-connector": "^1.1.7"
  }
}
```

# Install command:
```bash
npm install googleapis pusher-js tiktok-live-connector
```

Or with yarn:
```bash
yarn add googleapis pusher-js tiktok-live-connector
```
